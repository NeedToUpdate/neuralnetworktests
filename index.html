Neural Network Tests

All of these projects are made with just one library. It has no dependencies, other than self-written matrix and vector libraries. The drawing is all done by a rather simple div manipulation library, that was improved heavily with this future project

solving MNIST with a simple dense NN

This uses a model with 3 dense layers with sigmoid activation functions.

solving MNIST with a CNN

This model uses 2 convolution layers with relu and maxpools, fololwed by one dense layer. Much more effective than the nn above

Predicting a single word with a simple RNN

This model has a single reccurent layer, followed by a dense layer. It can learn one word basically.

Learning a whole alphabet with a LSTM

This model now uses a single LSTM layer to achieve learning much longer words and sentences than the model above.

word2vec trials

Using a simple dense layer, creates an encoder function. With 3 sentences, you can see the words grouping together with thei xy position or colour

Eaters with crossover mutations

The little dudes all have a simple 2 dense layer brain. Once they all die from hunger, the two best dudes have their brains mixed together, and become the parents of the next generation.

Smart Rockets

This was a an attempt to copy this project, after seeing it on Coding Train. The rockets can't see the target, they only know thier own speed, x and y position, and how much fuel they have left. Once they all die, the fittest gets cloned and mutated.
